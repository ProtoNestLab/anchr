# API Reference

Anchor SDK is split into three packages:

| Package                    | Description                                   |
| -------------------------- | --------------------------------------------- |
| [@anchor-sdk/core](./core) | Core types, client factory, adapter interface |
| [@anchor-sdk/vue](./vue)   | Vue 3 components and composables              |
| [@anchor-sdk/ui](./ui)     | Pre-built UI components                       |
